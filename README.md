# Family-Tree
Family tree and genealogy connections.

Developing with Swift 5 & SwiftUI with MVVM design pattern.

<h2>Done for now:</h2>

<b>Model</b>
1. Tree data structure (reusable).
2. Person data structure, we are using as a node value.
3. Login manager class.

<b>View</b>
1. Diagram (whole tree view).
2. MemberView (single node view).
3. Auth view(Sign Up, Sign In).
